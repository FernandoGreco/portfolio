Jornal Folha Oeste
Loja: jornalfolhaoeste.com.br

Neste site foi usado PHP, Mysql, CSS e JS. Não usei framework. Existe um login para usuário cadastrar, deletar notícia. No final da home existe um botão "carregar mais", neste local foi utilizada AJAX. No menu edições é apresentado as edições do jornal que são cadastradas pelo painel, ao cadastrar é criado um pasta (mkdir) pelo PHP com o número da edição. As notícas cadastradas, é salvo na tabela notícia o título, texto, endereço da imagem, e data atual. Para as url foi configurado o arquivo htaccess. Neste projeto foi criado um campo que permite o cliente fazer a edição pelo painel, foi criado "Ofertas Relampagos" como pode ser visto na imagem abaixo: Pelo painel o cliente escolhe a data que "Ofertas Relampagos" estará disponível na loja.
